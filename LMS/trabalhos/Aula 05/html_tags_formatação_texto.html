A tag <b> define um texto em negrito.

A tag <strong> define um texto importante.

A tag <em> define um texto enfatizado.

A tag <i> define um texto em itálico.

A tag <small> define um texto em menor tamanho que o padrão da página.

A tag <sub> define um texto subescrito.

A tag <sup> define um texto sobrescrito.

A tag <ins> define um texto inserido ou sublinhado.

A tag <del> define um texto excluído.

A tag <mark> define um texto marcado/realçado(highlighted).

A tag <br> força
uma quebra
de linha
e não precisa ser fechada.
